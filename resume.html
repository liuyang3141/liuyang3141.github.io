Job Scheduler Simulator
  • Simulates the Round Robin, non-preemptive Shortest Job First, and Shortest Remaining Time First scheduling algorithms.
  • Accepts a list of processes, their burst time, and arrival time in a text file.
  • For each scheduling algorithm, it will output the name of the scheduling algorithm used on the top. Each subsequent line includes an integer timestamp for when the process was handled, the name of the process, followed by the reason for a context switch. Lastly, it will output a table that contains the values for the process ID, turnaround time, and waiting time for each process, as well as the average for turnaround time and waiting time.
